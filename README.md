Project overview

Invisible Work Tracker

A tool meant to highlight the quiet everyday actions from people help make the earth better. Actions that usually go unnoticed but together make a difference.

Stack
-Front end (HTML, CSS, JS)
-Backend (java SpringBoot)
-Desktop app (javafx)
-Database(Mysql)

System Architecture 

The applications follows a client-server application.

-Frontend and desktop app act as clients.
They display dashboards, user stats, graphics,
logging in data.

-the clients  connect with the backend,
which handles data processing, users, authentication, logic. 

-the data is stored within a MySQL server
stores users, logged interactions, time, location data.

Features
-Log invisible work actions
-Show glowing blips at the location of each action
-Basic stats (total actions, categories used, recent activity)
-Choose a category (environment, community, personal, home, etc.)
-Store actions in MySQL
